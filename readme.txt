Get your customer's location by google map in your woocommerce checkout page

== Description ==
Use Google Map to get your customer's location in checkout page and view it in your admin panel.

- The easiest way to add map to your e-commerce website
- No coding required!
- Get customer's location in admin panel order page


== Installation ==
1. Search for Woo Google Map through 'Plugins > Add New'.
2. Find the plugin box of Woo Google Map and click on the 'Install Now' button.
3. Then activate the Woo Google Map plugin.
4. Go to Plugin settings page and add your Google map API key and default lat & long to display.
5. enjoy it! :)
